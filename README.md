# NASA Medical Inventory System

This system was made for the NASA Hunch Modified Medical Inventory System project. It combines facial recognition, pill recognition, inventory tracking, inventory analysis, and security all into one, unified system. 

## Features

- Facial Recognition Authentication
- Real-time Inventory Tracking
- Usage Analytics and Reporting
- AI-Powered Pill Recognition
- ESP32-Controlled Smart Lock
